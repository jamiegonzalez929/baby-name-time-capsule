How to read it

The dataset is compact by design, so the interaction stays quick on GitHub Pages.

  • The view shows the top 25 names for each year and sex.
  • The movement chart tracks only names that reached the top 10 in either year.
  • Percentages are shares of births for that sex in the selected year.
  • The mirrored source dataset in this project runs from 1880 through 2008.
